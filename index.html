隐私承诺

您的声音，
只属于您。

言墨不单独收集任何语音数据，语音识别通过 Azure Speech Service 完成，识别结果仅用于插入编辑器，不作任何存储或追踪。这不是一项功能，这是我们的底线。

常见问题

您可能想知道

如何配置 Azure 语音凭据？

打开应用设置，搜索 yanmo，填入您的 Azure Speech Service 密钥与区域即可。应用已内置演示凭据，安装后可直接试用，无需立刻配置。

支持哪些识别语言？

基于 Azure Cognitive Services，支持中文（普通话、粤语）、英文、日文等 100+ 语言，可在设置中自由切换。混合语言场景下可开启自动检测模式。

AI 润色功能需要额外费用吗？

AI 润色通过 Azure OpenAI 提供。使用自己的 Azure OpenAI 密钥时，费用按 token 实际消耗计算，通常极低。演示模式下内置配额可免费体验，超出后请配置自有密钥。

录音结果如何插入到编辑器？

识别完成后，文字会自动插入到当前光标所在位置，行为与普通键盘输入一致。同时支持通过命令面板（Ctrl/Cmd + Shift + P）调用「言墨：开始录音」随时触发。

自动润色和快捷短语保存在哪里？

所有数据以 JSON 文件形式保存在本地。数据不会上传到任何服务器，您可以随时备份或迁移。
